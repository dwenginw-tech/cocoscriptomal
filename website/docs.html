CocoScript Documentation

Getting Started

CocoScript is a compiled language that generates native x64 assembly. To get started, clone the repository and build the compiler:

git clone https://github.com/dwenginw-tech/cocoscriptomal.git
cd cocoscriptomal
dune build
./_build/default/src/driver/main.exe yourfile.coco

Syntax Basics

CocoScript uses a C-like syntax with semicolons and curly braces:

// Single-line comment
/* Multi-line
   comment */

print("Hello, World!");

Variables

Declare variables with let:

let x = 42;
let name = "CocoScript";
let result = x + 10;

Functions

Define functions with parameters and return values:

function add(a, b) {
    return a + b;
}

let sum = add(5, 3);
print(sum);

Functions support closures:

function makeCounter() {
    let count = 0;
    return function() {
        count = count + 1;
        return count;
    };
}

let counter = makeCounter();
print(counter()); // 1
print(counter()); // 2

Classes

CocoScript supports object-oriented programming with classes:

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        print("Hi, I'm " + this.name);
    }
}

let person = Person("Alice", 30);
person.greet();

Standard Library

CocoScript includes built-in libraries for common tasks:

Math Library

include "math.coco";

let result = Math.sqrt(16);
let power = Math.pow(2, 8);

String Library

include "string.coco";

let length = String.length("hello");
let upper = String.toUpper("hello");
